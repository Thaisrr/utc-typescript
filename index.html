Typescript

Typescript est une surcouche de Javascript, développée par Microsoft.

Il apporte des fonctionnalités supplémentaires, dont la notion de typage.

C'est un langage transpilé : compilé en Javascript.

TSC : compilateur typescript, s'installe avec npm :

npm i -g typescript

Node.JS

Permet d'exécuter du Javascript côté serveur.

Se base sur Common JS, et utilise le moteur Javascript de Chrome V8.

NPM : Gestionnaire de librairie de Node.

Permet de gérer des librairies en ligne de commande.

Pour un projet, ou sur le PC.

Associé a un fichier Package.json

Commandes NPM :

  • npm i (install ) [librairie] => installe un librairie dans le projet
  • npm r ( remove ) [librairie] => supprime une librairie
  • npm i -g [librairie] => installe sur le PC
  • npm i --save-dev [librairie] => installe une librairie de dev
  • npm list -g => liste les librairies globales
